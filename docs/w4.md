# Week 4 Report

## Problem Statements

### Clarify the 5 Ws

**Five Ws**
- Who — Who is using the site  
- What — What is the site offering/doing, what should the main focus be  
- Where — Online  
- When — ASAP, later?  
- Why — Why create/redesign the site  

---

## Feature List

Scope — what are you going to do/implement for this website redesign  

**Examples:**
- Update branding (colors, typography)  
- Redesign home page layout  
- Redesign specific page layouts  
- Redesign menu  

---

## Ideation

Generate as many possible solutions to the problem as you can (non-judgemental)

**Techniques**
- Brainstorming  
- Worst Possible Idea  
- Crazy 8s (quick wireframe sketches)  

---

## Information Architecture (IA)

Organizing and labelling content to help users find information  

**Includes:**
- Navigation (menu type, number of menus, hierarchy)  
  - Deep → few top-level links, many nested links  
  - Shallow → many top-level links, fewer nested links  
- Labelling (language/style used)  
- Content inventory (list of all pages/content)  
- Sitemap (visual structure of content)  
- Wireframes (layout sketches of pages)
