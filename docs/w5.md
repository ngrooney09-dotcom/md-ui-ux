# Week 5 Report

## What I Learned in ACIT 2811 (UX/UI)

### Information Architecture

This week focused on how to organize and structure content so users can easily navigate and understand a website or application.

### Navigation Systems

Navigation systems allow users to move through a website or app.

- Includes menus, links, and navigation bars  
- Should be simple and consistent  
- Helps users find information quickly 

### Labelling Systems

Labelling systems refer to the names used for menus, buttons, and sections.

- Should be clear and easy to understand  
- Uses familiar and common terms  
- Avoids confusing or vague wording 

### Sitemap

A sitemap is a visual structure of all pages in a website or application.

- Shows how pages are connected  
- Helps organize content  
- Supports better navigation design  

### What we got out of class

- Understood the importance of clear navigation systems  
- Improved ability to label sections in a user-friendly way
